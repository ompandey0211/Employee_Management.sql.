\# Employee Management System Using SQL



\## Description

This project demonstrates an Employee Management System developed using MySQL. It includes database creation, table design, data insertion, and SQL queries using SELECT, WHERE, GROUP BY, HAVING, JOINs, aggregate functions, and subqueries for employee and department analysis.



\## Tools Used

\- MySQL

\- SQL

\- Git

\- GitHub



\## Features

\- Employee and Department tables

\- Primary Key \& Foreign Key relationships

\- CRUD operations

\- Aggregate Functions

\- GROUP BY \& HAVING

\- JOIN Queries

\- Subqueries

\- Business Analysis
